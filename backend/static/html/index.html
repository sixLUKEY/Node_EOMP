Home

HTTP Methods Path Description
GET /(Home page) Takes you to the home page

Users Table

HTTP Methods Path Description
GET /users displays a list of users
GET /users/:id displays a specific user based on id
POST /users Creates a new user/register a new user
POST /logins Checks if user information is correct the allows you to log in
PUT /users/:id Updates an existing user
DELETE /users/:id Deletes a user

Products Table

HTTP Methods Path Description
GET /products Displays a list of products
GET /products/:id Displays a single user by id
POST /products Adds a new product
PUT/PATCH /products/:id Updates a products details
DELETE /products/:id deletes a product from the database
